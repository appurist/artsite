üìã Manual Setup Instructions

Go to your Appwrite Console and create these manually:

1. Create Database

  • Go to Database section
  • Create new database with ID: artsite
  • Name: "Art Gallery Database"

2. Create "artworks" Collection

  • Inside the database, create collection with ID: artworks
  • Add these attributes:
    • title - String, Size: 255, Required: ‚úì
    • description - String, Size: 2000, Required: ‚úó
    • medium - String, Size: 255, Required: ‚úó
    • dimensions - String, Size: 255, Required: ‚úó
    • year_created - Integer, Required: ‚úó
    • price - String, Size: 255, Required: ‚úó
    • tags - String, Size: 1000, Required: ‚úó
    • image_id - String, Size: 255, Required: ‚úì
    • filename - String, Size: 255, Required: ‚úó
    • created_at - DateTime, Required: ‚úì
  • Set permissions: Read: Any, Write: Users

3. Create "settings" Collection

  • Create collection with ID: settings
  • Add these attributes:
    • key - String, Size: 255, Required: ‚úì
    • value - String, Size: 5000, Required: ‚úì
  • Create index: key_unique on key field (unique)
  • Set permissions: Read: Any, Write: Users

4. Create Storage Bucket

  • Go to Storage section
  • Create bucket with ID: images
  • Name: "Images"
  • File extensions: jpg,jpeg,png,gif,webp
  • Set permissions: Read: Any, Write: Users

5. Platform Configuration

  • Go to Settings ‚Üí Platforms
  • Add Web Platform
  • Name: "Art Gallery Dev"
  • Hostname: localhost:5173

6. Create Admin User

  • Go to Auth ‚Üí Users
  • Create new user with email/password
  • Note the credentials for testing
